## About

**HISBA** is a professional Product Cost Calculator designed to help businesses calculate product costs accurately and efficiently.

The application supports multiple industries including:

- Restaurants
- Cafés
- Bakeries
- Manufacturing
- Retail
- E-commerce
- Handmade Products
- Any business that requires professional cost analysis

## Run Locally

### Prerequisites

- Android Studio (Latest Version)

---

### Installation

1. Open **Android Studio**
2. Select **Open Existing Project**
3. Choose this project folder
4. Let Android Studio finish indexing
5. Create a file named:

```
.env
```

6. Add your API Key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

7. Build the project

8. Run on Emulator or Android Device
